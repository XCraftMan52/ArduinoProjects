# Simple - Press Button Alternating LEDS
<br>

## Description
A simple Arduino program to turn on alternating LEDs when one button is pressed.

<br>

## Parts List

| Name     	| Quantity 	| Component      	|
|----------	|----------	|----------------	|
| U1       	| 1        	| Arduino Uno R3 	|
| S1       	| 1        	| Pushbutton     	|
| R1<br>R2<br>R3| 3        	| 1k Resistor    	|
| D1<br>D2| 2        	| Red LED        	|
