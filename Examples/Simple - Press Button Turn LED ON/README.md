# Simple - Press Button Turn LED On
<br>

## Description
A simple Arduino program to constantly turn on a LED when a button is pressed.

<br>

## Parts List

| Name     	| Quantity 	| Component      	|
|----------	|----------	|----------------	|
| U1       	| 1        	| Arduino Uno R3 	|
| S1       	| 1        	| Pushbutton     	|
| R1<br>R2| 2        	| 1k Resistor    	|
| D1| 1        	| Red LED        	|
